fingerprintRaw

Get the raw fingerprint for a request.

exports.fingerprintRaw = (req) -> fingerprint = {}

Invoke hook fingerprint. Allows a package to specify unique keys for this request, e.g. IP address, session ID, etc. Implementations take a request object as the only parameter. The request parameter may be null.

for keys in pkgman.invokeFlat 'fingerprint', req continue unless keys? fingerprint[key] = value for key, value of keys fingerprint
