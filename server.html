Server

The server application entry point. We load the configuration, invoke the initialization hooks, and listen for signals and process exit.

pkgman.invokeFlat 'initialize', config # After initialization. ).done(

Invoke hook ready. Invoked after the server is initialized and ready.

-> pkgman.invoke 'ready' (error) -> defaultLogger.error errors.stack error # Rethrow any error. throw error )

Do our best to guarantee that hook processExit will always be invoked.
