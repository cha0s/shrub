Client application entry point.

Definition of the main top-level Angular module, with dependency gathering and config/run hook invocation.

Top-level module.

angular.module 'shrub', ['shrub.core']

Include core Angular dependencies.

coreDependencies = []
coreDependencies.push 'ngRoute'
coreDependencies.push 'ngSanitize'

packageDependencies will be automatically generated and populated by Grunt. Packages implement the hook angularPackageDependencies to specify their 3rd-party module dependencies.
