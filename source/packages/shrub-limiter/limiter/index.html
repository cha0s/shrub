Limiter

Provides methods to tally scores, and compare them against a threshold of time. The limiter package implements hook shrubRpcRoutesAlter to allow RPC routes to limit consumers to a specified number of requests per time period. For instance, by default the user login route limits the number of logins a user may attempt to 3 every 30 seconds.

module.exports = class Limiter

Limiter.threshold

Expose a factory method for constructing Threshold instances. A Threshold is defined like: var Limiter = require('shrub-limiter').Limiter; Limiter.threshold(4).every(20).minutes(); Which means that the threshold represents the allowance of a score of 4 to accumulate over the period of 20 minutes. If more score is accrued during that window, then the threshold is said to be crossed.

  @threshold: (score) -> new ThresholdBase score

Limiter#_largest

Find the largest result from a group of results.

  _largest: (keys, index) ->
    Promise.all(
      @limiter[index] key for key in keys

    ).then (reduction) ->
      reduction.reduce ((l, r) -> if l > r then l else r), -Infinity
