Implements hook appRun.

  registrar.registerHook 'appRun', -> [
    '$window', 'shrub-rpc'
    ($window, rpc) ->

Hang up the socket unless it's the local (Node.js) client.

      unless $window.navigator.userAgent.match /^Node\.js .*$/
        rpc.call 'shrub.hangup'
