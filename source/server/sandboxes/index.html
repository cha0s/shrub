Sandboxes

This module provides a method for creating sandboxed DOMs (using jsdom). It augments the DOM with a functional WebSocket using ws, and generally makes spinning up arbitrary DOM contexts a pleasant breeze.

config = require 'config'
Promise = require 'bluebird'
WebSocket = require 'ws/lib/WebSocket'

errors = require 'errors'
logging = require 'logging'

{jsdom} = require 'jsdom'

Hax: Fix document.domain since jsdom has a stub here.

{HTMLDocument} = require 'jsdom/lib/jsdom/living'
Object.defineProperties(
  HTMLDocument.prototype
  domain: get: -> 'localhost'
)

This class is responsible for creating and cleaning up DOMs, and provides some methods to inspect the state of the document.
