A route is defined like:

  • controller: An annotated function which will be injected.

  • template: A template string.

  • title: A string which will be set as the page title.

Split the path into the corresponding classes, e.g.

foo/bar/baz -> class="foo foo-bar foo-bar-baz"

$rootScope.$watch (-> $location.path()), -> parts = $location.path().substr(1).split '/' parts = parts.map (part) -> part.replace /[^_a-zA-Z0-9-]/g, '-' classes = for i in [1..parts.length] parts.slice(0, i).join '-' $rootScope.pathClass = classes.join ' '
