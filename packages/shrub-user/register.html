register

Register a user in the system.

  • (string) name - Name of the new user.

  • (string) email - Email address of the new user.

  • (string) password - The new user's password.

exports.register = (name, email, password) -> {User} = schema.models user = new User name: name, iname: name.toLowerCase()

Encrypt the email.

crypto.encrypt(email.toLowerCase()).then((encryptedEmail) -> user.email = encryptedEmail

Set the password encryption details.

crypto.hasher plaintext: password ).then((hashed) -> user.plaintext = hashed.plaintext user.salt = hashed.salt.toString 'hex' user.passwordHash = hashed.key.toString 'hex'

Generate a one-time login token.

crypto.randomBytes 24 ).then (token) -> user.resetPasswordToken = token.toString 'hex' user.save()
