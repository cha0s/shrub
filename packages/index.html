Package overview

Packages are how Shrub organizes functionality. Packages may be provided for the server or the client (or both).

This page provides a listing of packages in this project, along with a short description of the functionality they provide.

Client-side

shrub-angular — Communicate with Angular on the server

Implements hooks

appRun

shrub-core — Core client functionality.

Coordinate various core functionality.

Implements hooks

aHrefSanitizationWhitelist, appConfig, appRun, route

Invokes hooks

aHrefSanitizationWhilelist, route

shrub-example — Example

shrub-example/client/about — Example - About page

Implements hooks

route

shrub-example/client/home — Example - Home page

Implements hooks

appConfig, route

shrub-form — Form processing

Define a directive for Angular forms, and a service to cache and look them up later.

Implements hooks

directive, service

Invokes hooks

formAlter, formFormIdAlter, formWidgets

shrub-form/client/widget/checkbox — Form - checkbox

Implements hooks

directive, formWidgets

shrub-form/client/widget/checkboxes — Form - checkboxes

Server-side

shrub-angular — Angular on the server

A sandboxed version of Angular, for clients lacking JS.

Implements hooks

endpoint, httpMiddleware, bootstrapMiddleware, packageSettings

Invokes hooks

angularNavigationMiddleware

shrub-assets — Assets management

Gather, build, and serve assets defined by packages.

Implements hooks

clearCaches, assetMiddleware, gruntConfig, packageSettings

Invokes hooks

assetMiddleware

shrub-assets/angular — Angular assets

Implements hooks

assetMiddleware

shrub-assets/jquery — jQuery assets

Implements hooks

assetMiddleware

shrub-audit — Auditing

Track users through the request. This can be used to weed out bad behavior, for analytics, or anything else.

Implements hooks

httpMiddleware, socketAuthorizationMiddleware

shrub-audit/fingerprint — Fingerprint class

Invokes hooks

fingerprint

shrub-config — Configuration

Gathers configuration and serves it as an Angular module for clients.

Implements hooks

assetMiddleware, httpMiddleware

Invokes hooks

config

shrub-core — Core server functionality

Coordinate various core functionality.

Implements hooks

config, fingerprint, httpMiddleware, packageSettings, replContext, socketAuthorizationMiddleware

shrub-example

Implements hooks

notificationQueue

shrub-example/about — Example - About page

Define a route to access the README.md page.

Implements hooks

httpRoutes

shrub-form — Form processing

Handle form and method parsing, and submission of POST'ed data into the Angular sandbox.

Implements hooks

angularNavigationMiddleware, httpMiddleware
