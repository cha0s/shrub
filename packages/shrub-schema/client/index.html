Schema::definePackageModels

Invoke hook models. Allows packages to create models in the database schema.

pkgman.invoke 'models', @

Invoke hook modelsAlter. Allows packages to alter any models defined.

pkgman.invoke 'modelsAlter', @models, @

Schema::resourcePaths

Translate model names to REST resource/collection paths. 'CatalogEntry' -> ['catalog-entry', 'catalog-entries']

Schema::resourcePaths = (name) -> resource = i8n.dasherize(i8n.underscore name).toLowerCase() resource: resource collection: i8n.pluralize resource
