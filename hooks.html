Hook overview

Shrub implements message passing between packages through a hook system. Hooks may be invoked with pkgman.invoke(), and are implemented in packages by exporting pkgmanRegister.

For instance, if we are implementing a package and want to implement the httpListening hook, our code would look like:

exports.pkgmanRegister = (registrar) ->

    registrar.registerHook 'httpListening', ->

        # Your code goes here...
